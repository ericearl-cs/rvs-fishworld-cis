# RVS Fishworld CIS

Current local backup snapshot of the Java CIS project.

## Run

1. Copy `config.properties.example` to `config.properties`
2. Update the database values for your local environment
3. Run `compile.bat`
4. Run `run.bat`

## Included

- `src/`
- `sql/`
- `docs/`
- `tools/`
- build/run scripts

## Excluded

- `config.properties`
- `out/`
- `logs/`
- `recovery_staging/`
- editor-specific folders
